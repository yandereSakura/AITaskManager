# AI Task Manager

Приложение для управления задачами.

## Стек технологий

- Frontend: React + JavaScript
- Backend: Node.js
- База данных: PostgreSQL
- Дополнительно: Python
- Git для контроля версий

## Что умеет приложение

- Создавать новую задачу
- Смотреть список всех задач
- Менять статус задачи (новая / в процессе / выполнена)
- Удалять задачу

## Из чего состоит проект

**backend** — серверная часть на Node.js. Здесь находится логика работы с базой данных и API.

**frontend** — клиентская часть на React. Здесь интерфейс, который видит пользователь.

**export_tasks.py** — скрипт на Python, который выгружает все задачи из базы данных в CSV-файл.

## Как запустить проект

### Шаг 1. База данных

Установи PostgreSQL и создай базу данных с именем `taskmanager`.

Затем открой файл `backend/db/init.sql` и скопируй весь его текст в Query Tool PostgreSQL и нажми Execute.

### Шаг 2. Backend (сервер)

Открой терминал в папке `backend` и выполни:
npm install
node index.js

Сервер запустится на адресе http://localhost:5000

Файл `.env` в backend не загружен на GitHub, поэтому  его нужно создать со своими данными:
Нужно создать файл `.env` внутри папки `backend` и указать там данные для подключения к базе данных:
DB_HOST=localhost
DB_PORT=5432
DB_NAME=taskmanager
DB_USER=postgres
DB_PASSWORD=ваш_пароль_от_postgresql
PORT=5000

### Шаг 3. Frontend (сайт)

Открой терминал в папке `frontend` и выполни:
npm install
npm start

Сайт откроется автоматически на http://localhost:3000

### Шаг 4. Экспорт задач в CSV (Python)

Установи библиотеку для подключения к базе данных:
pip install psycopg2-binary

Запусти скрипт:
python export_tasks.py

Появится файл с задачами в формате CSV.
